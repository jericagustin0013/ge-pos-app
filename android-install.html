Install Graphique Eloise on Android

Open the app in Chrome or Edge on Android. When eligible, tap the in-app Install button or use the browser menu and choose Add to Home screen / Install app.

Important: if you host with XAMPP and open the app on your phone using a Wi-Fi IP address such as http://192.168.1.10/..., Android Chrome treats that as normal HTTP. The real PWA install prompt requires HTTPS, except for localhost on the Android device itself.

  1. Host this folder on HTTPS, or open it from localhost on the Android device.
  2. Visit printing-pos.html in Chrome or Edge.
  3. Wait until the status shows Offline ready.
  4. Tap Install, then confirm.
  5. Launch Graphique Eloise from the Android home screen.

The app stores POS data in the phone/tablet browser database and keeps the app shell cached for offline use after the first successful load.
