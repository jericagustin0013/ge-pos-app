Install PrintDesk POS on Android

Open the app in Chrome or Edge on Android. When the app is eligible, tap the in-app Install button or use the browser menu and choose Add to Home screen / Install app.

Important: if you host with XAMPP and open the app on your phone using a Wi-Fi IP address such as http://192.168.1.10/..., Android Chrome treats that as normal HTTP. The real PWA install prompt requires HTTPS, except for localhost on the Android device itself.

  1. Host the outputs folder on HTTPS, or open it from localhost on the Android device.
  2. Visit printing-pos.html in Chrome or Edge.
  3. Wait until the status says Offline ready.
  4. Tap Install, then confirm.
  5. Launch PrintDesk from the Android home screen.

For XAMPP testing without HTTPS, use Chrome menu then Add to Home screen. That creates a shortcut, but Chrome may not treat it as a fully installable offline app until the site is served over HTTPS.

The app stores POS data in the phone/tablet browser database and keeps the app shell cached for offline use after the first successful load.
